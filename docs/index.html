サービス概要

襖張替え・補修

破れや汚れの張替えはもちろん、建て付け調整や引手交換などの補修にも対応しております。 和室に合う定番柄からモダンなデザインまで、現地で見本帳をご覧いただきながらお選びいただけます。

障子張替え・補修

破れた障子の張替えや桟の補修、滑り調整まで丁寧に対応いたします。 普通紙・強化紙・破れにくいタイプなど、用途に合わせて最適な障子紙をご提案します。

選ばれる理由

技術力

・経験豊富な職人が、素材特性と部屋環境を見極めて施工します。

・傷みの大きい襖や障子でも、補修・張り替えによって見違えるほど美しく仕上げます。

・「もう新調しかないかな」と思われる状態でも、費用を抑えながら見栄えよく整えられる場合が多くあります。

迅速対応

・地域密着だからこそ迅速に対応。

・ご相談から施工完了まで、内容にもよりますが基本3〜7日程度で仕上げております。

・お急ぎのご相談にも柔軟に対応いたします。

和室の印象を、今すぐ整える。

写真送付だけの簡易見積りにも対応しています。
